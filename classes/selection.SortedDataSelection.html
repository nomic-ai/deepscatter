applyToAllLoadedTiles

  • applyToAllLoadedTiles(): Promise<void>

  • Ensures that the selection has been evaluated on all tiles loaded in the deeptable. This is useful if, for example, your selection represents a search, and you are zoomed in on one portion of the map; this will immediately execute the search (subject to delays to avoid blocking the main thread) on all tiles that have been fetched even if out of the viewport.

    Resolves upon completion.

    Returns Promise<void>

get

  • get(i?): StructRowProxy

  • Returns the nth element in the selection. This is a bit tricky because the selection is stored as a list of tiles, each of which has a list of matches. So we have to iterate through the tiles until we find the one that contains the nth match, then iterate through the matches in that tile until we find the nth match.

    Parameters

    • i: number = undefined

      the index of the row to get. If less than zero, will return the element counting from the end of the selection.
